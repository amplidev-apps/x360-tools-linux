
# Melhorias de UX: Gerenciador de Downloads Profissional

Para elevar o nível da aplicação, implementaremos funcionalidades comuns em gerenciadores de elite (Steam, JDownloader):

## Menu de Contexto (Botão Direito no Item)
- **Abrir Pasta de Destino**: Localiza o arquivo baixado no explorador do Linux.
- **Ver detalhes no Catálogo**: Navega de volta para a descrição do jogo.
- **Remover/Cancelar**: Gerenciamento direto pelo menu.

## Barra de Ferramentas Global (Downloads)
- **Limpar Concluídos**: Remove da lista todos os itens que já finalizaram.
- **Pausar/Retomar todos**: Integração de UI para controle em lote.

## Refinamento Visual
- **Badges de Status**: Etiquetas coloridas (BAIXANDO, CONCLUÍDO, ERRO).
- **Tooltips dinâmicos**: Informações extras ao passar o mouse.
