<a name="english"></a>
## 🇬🇧 English

**x360 Tools for Linux** is a premium, all-in-one suite designed for Xbox 360 and Xbox Classic enthusiasts. Built with a high-performance Python backend and a stunning Flutter v2.0 interface, it provides a seamless experience for managing your console content, installing homebrew, and downloading games directly to your PC or Xbox-formatted drives.

### Key Features
*   **Premium Freemarket Store**: Browse and download thousands of Xbox 360 and Classic titles.
*   **Advanced Downloads Manager**: Real-time multi-phase tracking (Downloading, Extracting, Converting, Installing).
*   **Horizon Injector**: Inject STFS content, profile pictures, and manage console files.
*   **Xbox Classic Support**: Automatic ISO to Games conversion.
*   **Title Update Manager**: Easily find and install the latest updates for your games.
*   **Native Linux Performance**: Optimized for modern distributions with `xdg-open` integration.

### How to Run
1. Ensure you have `python3` and `flutter` installed.
2. Run the initialization script:
   ```bash
   ./run_flutter.sh
   ```
